Salva is a fluids simulation library written using the [Rust programming language](https://www.rust-lang.org/).
It targets applications requiring soft real-time physics like video games, animation.

Salva is free and open-source, released under the Apache 2.0 license. It is developed by the [Dimforge](https://dimforge.com)
open-source company. You can support us by sponsoring us on [GitHub sponsor](https://github.com/sponsors/dimforge).
